About this site

This site is directly linked to my GitHub repository, and is hosted by GitHub through GitHub Pages. The stylesheet template am I currently using is provided by Bulma, a free open-source CSS library which comes with only 1 CSS file. Bulma configures the Light/Dark theme depending on your device settings. A manual switch is planned, and will be online soon. If you still have any questions or concerns, or would like to report a problem with this site, please reach out to me by the contacts listed in the contacts page. Thank you!
